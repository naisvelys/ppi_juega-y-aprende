DESCRIPCION

Este proyecto está ingeniado para los niños de preescolar y primero, ya que estos son los primeros grados de ciclo escolar de un niño y desde estos se puede ir implementando algunas cosas con lectura y escritura. Como son grados diferentes obviamente no se implementarán las mismas cosas, cada niño y dependiendo su grado tiene una mentalidad diferente. El proyecto está basado en que se implementarán juegos didácticos en cada práctica de aprendizaje, se enseñará como suenan las letras solas y acompañadas de otras letras también, como se escriben, que sonidos producen cada letra, como se juntan las letras para lograr crear palabras. Tendrá una especie de actividad para lograr ver si están logrando captar lo aprendido. Teniendo en cuenta sus edades y su nivel de aprendizaje, habrá un aprendizaje diferente en cada grado
